# OverPanel

> Современная панель управления для Rust-серверов с поддержкой мультипроектности, гибкой системой ролей и инструментами для профессиональной модерации.

---

## Что такое OverPanel?

OverPanel — это современная веб-панель, созданная для владельцев Rust-серверов и крупных игровых проектов. Мы старались сделать инструмент, который не перегружает интерфейс лишними элементами, но при этом предоставляет всё необходимое для эффективного управления серверами, администрацией и игроками.

Панель одинаково хорошо подходит как для небольшого сообщества с одним сервером, так и для сети из десятков серверов.

---

## Основные возможности

### 🌍 Мультипроектность и мультисерверность

В одном аккаунте можно управлять сразу несколькими проектами, а внутри каждого проекта — любым количеством серверов.

Добавление нового сервера занимает всего несколько секунд: достаточно выполнить команду

```bash
op.pair <код>
```

после чего сервер автоматически появится в панели.

Поддерживаются как **Carbon**, так и **Oxide/uMod**.

---

### 👥 Полностью настраиваемые роли

В OverPanel нет жёсткой привязки прав к определённым уровням доступа.

По умолчанию доступно десять уровней ролей, однако при необходимости можно создать собственные роли и самостоятельно настроить для них каждое разрешение.

Например, если младшему модератору нужен доступ к функции, которая обычно доступна только главному администратору, её можно выдать отдельно, не открывая остальные возможности.

---

### 🎙 Проверки игроков с голосовым сопровождением

Во время проверки игрок получает не только текстовые инструкции через CUI, но и голосовые сообщения прямо в игре.

OverPanel автоматически определяет язык клиента (русский или английский) и отображает как текст, так и озвучивание на нужном языке.

Для воспроизведения голоса используется специальный виртуальный игрок, благодаря чему всё выглядит максимально естественно.

---

### 🛡 Собственный чит-чекер

В состав проекта входит отдельное desktop-приложение для проверки игроков.

Во время проверки анализируется более 30 различных категорий, включая:

- читы;
- DLL-инжекты;
- процессы;
- изменения реестра;
- автозагрузку Windows;
- VPN-клиенты;
- драйверы;
- следы системных модификаций;
- и многое другое.

Также приложение собирает HWID, что значительно повышает эффективность поиска мультиаккаунтов.

---

### 🔗 Борьба с мультиаккаунтами

OverPanel запоминает связи между аккаунтами навсегда.

Если игрок уже получил блокировку на связанном аккаунте, система сможет автоматически обнаружить это и применить необходимые ограничения.

---

### 🚫 Общая сеть банов

несколько проектов могут обмениваться информацией о заблокированных игроках.

Это позволяет быстрее реагировать на нарушителей и значительно усложняет обход блокировок между серверами.

---

### 💻 Полноценный RCON

В панели встроена полноценная консоль с выводом событий в режиме реального времени.

Для каждой роли можно отдельно определить список запрещённых команд. Например, ограничить доступ к `c.*`, `oxide.*` и другим критически важным командам для всех сотрудников, кроме владельца проекта или главного администратора.

---

### 🔒 Защита от злоупотреблений

OverPanel помогает защитить сервер не только от нарушителей, но и от возможных злоупотреблений со стороны администрации.

Можно настроить правила, при которых система автоматически снимет права с администратора, если за определённый промежуток времени он выдаст слишком большое количество банов, мутов или других наказаний.

При необходимости можно одной операцией отменить все наказания, выданные конкретным администратором за выбранный период.

---

### 📊 Публичный API

Панель предоставляет API только для чтения данных.

С его помощью можно получать:

- список банов;
- статистику;
- информацию об онлайне;
- данные о проектах.

API отлично подходит для интеграции с собственными сайтами, лаунчерами и сторонними сервисами.

## SaaS-модель

OverPanel распространяется по подписке и доступен в трёх тарифах:

| Тариф | Описание |
|--------|----------|
| **Basic** | Для небольших проектов |
| **Pro** | Для активно развивающихся серверов |
| **Enterprise** | Для крупных сетей с индивидуальными условиями |

Для новых пользователей доступен **бесплатный пробный период 14 дней**. Этого времени достаточно, чтобы полноценно оценить возможности панели, протестировать основные функции и понять, подходит ли OverPanel именно вашему проекту.

Тариф **Enterprise** дополнительно включает индивидуальные лимиты, собственный поддомен и возможность кастомизации под конкретный проект.

## Ограничения использования

OverPanel предназначен исключительно для работы с серверами, использующими **официальную версию Rust**.

Использование панели на пиратских серверах, а также её модификация или адаптация для поддержки неофициальных клиентов **запрещены**.

Все обновления, новые функции и техническая поддержка предоставляются только для проектов, работающих на лицензионной версии игры. Совместимость с пиратскими клиентами не разрабатывается и не планируется.

При использовании панели на неофициальных серверах ваш проект может быть заблокирован.
При заходе игроков с пиратского клиента игры у вас будет предупреждение.
