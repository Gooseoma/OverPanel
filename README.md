# OverPanel

**Панель управления для Rust-серверов**

Один аккаунт для всех твоих серверов, гибкие права для админов, проверки читеров с голосом и свой чит-чекер.

## Что это такое

OverPanel — это панель, через которую владелец Rust-серверов управляет своими проектами: выдаёт права администрации, банит читеров, следит за консолью сервера и смотрит статистику. Подходит и для одного маленького сервера, и для сети из десятков серверов сразу.

Мы старались не захламлять интерфейс — только то, что реально нужно для управления сервером и командой.

## Что умеет панель

### 🌍 Все твои серверы в одном месте

В одном аккаунте можно завести сколько угодно проектов, а в каждом проекте — сколько угодно серверов. Подключить новый сервер очень просто: пишешь в консоли одну команду —

```
op.pair <код>
```

— и сервер сразу появляется в панели. Работает и на **Carbon**, и на **Oxide/uMod**, менять фреймворк не нужно.

### 👥 Права под любую команду

По умолчанию есть 10 готовых уровней доступа, но их можно не использовать — роли создаются с нуля, а каждое право включается или выключается отдельным переключателем. Например, можно дать младшему модератору одну конкретную функцию, которая обычно есть только у главного админа, и при этом не открывать ему всё остальное.

### 🎙 Проверка читеров с голосом

Когда админ проверяет игрока, тот видит текст на экране (через CUI) и одновременно слышит голос прямо в игре — звук идёт как будто от игрока, который стоит рядом. Панель сама определяет, на каком языке говорить с игроком — русском или английском.

### 🛡 Свой чит-чекер

В комплекте отдельная программа для компьютера игрока. Она проверяет больше 30 разных вещей: читы, подозрительные DLL, реестр Windows, автозагрузку, VPN, драйверы и так далее. Заодно собирает уникальный отпечаток компьютера (HWID) — это помогает находить вторые аккаунты нарушителей.

### 🔗 Поиск вторых аккаунтов и баны

Панель запоминает, какие аккаунты связаны между собой, и не забывает это никогда. Если у одного из связанных аккаунтов есть активный бан — второй банится автоматически. А ещё разные проекты могут по подписке делиться друг с другом списками забаненных, чтобы нарушителю было сложнее просто перейти на другой сервер.

### 💻 Консоль сервера прямо в браузере

Не нужно открывать отдельную программу для RCON — есть встроенная консоль, которая показывает всё, что происходит на сервере, в реальном времени. При этом для каждой роли можно ограничить список доступных команд — например, закрыть опасные команды вроде `c.*` и `oxide.*` для всех, кроме владельца.

### 🚨 Защита от нечестных админов

Можно настроить правило: если админ за короткое время выдал слишком много банов или мутов — панель автоматически снимет с него права. А если нужно всё отменить, можно одним действием убрать сразу все наказания конкретного админа за нужный период.

### 📊 Открытый API

Есть API, из которого можно получать данные (только на чтение) — списки банов, статистику, количество игроков онлайн. Удобно, если хочешь показать эти данные на своём сайте или в лаунчере.

## Тарифы

Панель работает по подписке, тарифов три:

| Тариф | Кому подходит |
|---|---|
| **Basic** | Небольшим проектам |
| **Pro** | Серверам, которые активно растут |
| **Enterprise** | Крупным сетям — свои лимиты и отдельный поддомен |

Есть бесплатный пробный период — можно спокойно попробовать панель на своём сервере перед покупкой.

## Какие платформы поддерживаются

- ✅ Carbon
- ✅ Oxide / uMod

---

## Как начать

1. Регистрируешься и создаёшь проект в панели.
2. Ставишь плагин OverPanel на сервер (Carbon или Oxide/uMod).
3. Выполняешь в консоли сервера команду с кодом, который панель выдаст при создании сервера:

   ```
   op.pair <код>
   ```

4. Сервер появляется в панели — дальше настраиваешь роли и права под свою команду.

---

## Про пиратские сборки сервера

Панель рассчитана на официальную версию Rust Dedicated Server. Пиратские и модифицированные сборки использовать не запрещено, но если что-то сломается именно из-за них — поддержку мы не окажем.
