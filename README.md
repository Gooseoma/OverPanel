# OverPanel

<p align="center">
  <b>Современная панель управления для Rust-серверов.</b><br>
  Минималистичная, быстрая и созданная для проектов любого масштаба — от одного сервера до целой сети.
</p>

---

## Почему OverPanel?

**OverPanel** — это современная веб-панель для управления Rust-серверами, созданная с упором на удобство, скорость работы и отсутствие перегруженного интерфейса.

Вместо десятков сложных меню вы получаете лаконичную панель, в которой легко ориентироваться даже при управлении несколькими проектами и десятками серверов.

---

# ✨ Основные возможности

## 🌍 Мультипроектность и мультисерверность

Управляйте всеми своими проектами из одного аккаунта.

### Возможности

- Один аккаунт → несколько проектов
- Один проект → несколько серверов
- Быстрое переключение между серверами
- Общая система ролей внутри проекта
- Поддержка **Carbon** и **Oxide/uMod**

### Подключение сервера

Для подключения нового сервера достаточно выполнить одну команду в консоли:

```bash
op.pair <код>
```

Никакой ручной настройки конфигураций.

---

# 👥 Гибкая система ролей

Полностью настраиваемая система прав без жёстких ограничений.

### Возможности

- 10 уровней доступа по умолчанию
- Создание собственных ролей с нуля
- Настройка каждого разрешения отдельно
- Управление правами через удобные тумблеры
- Нет привязки к фиксированным уровням

Например, можно выдать **младшему модератору** доступ к одной конкретной функции, которая обычно доступна только **главному администратору**, не открывая остальные права.

# 🎙 Проверки с голосовым сопровождением

Совершенно новый подход к проведению проверок игроков.

Во время проверки игрок увидит:

- CUI-оверлей прямо в игре
- Автоматический перевод текста
- Голосовое озвучивание сообщений
- Автоматическое определение языка клиента (RU / EN)

Для воспроизведения голоса используется специальный виртуальный игрок, находящийся рядом с проверяемым, благодаря чему озвучивание звучит максимально естественно.

---

# 🛡 Собственный чит-чекер

В состав OverPanel входит отдельное desktop-приложение для проверки игроков.

Во время проверки приложение анализирует более **30 категорий**, включая:

- Чит-программы
- DLL-инжекты
- Запущенные процессы
- Изменения реестра
- Автозагрузку Windows
- VPN-клиенты
- Подозрительные драйверы
- Следы системных модификаций
- и многое другое.

Также приложение собирает **HWID**, что позволяет значительно эффективнее выявлять мультиаккаунты и попытки обхода блокировок.

# 🔗 Система мультиаккаунтов

OverPanel навсегда запоминает связи между игровыми аккаунтами.

### Возможности

- Хранение истории связей
- Привязка по HWID
- Автоматическое определение связанных аккаунтов
- Автоматический бан при наличии активной блокировки на связанном аккаунте
- Защита от обхода наказаний

# 🚫 Общая сеть банов

*(Доступно по подписке)*

Несколько проектов могут обмениваться информацией о нарушителях.

### Что это даёт

- Общая база банов
- Автоматическая синхронизация
- Быстрое обнаружение рецидивистов
- Защита сообщества сразу на нескольких проектах

---

# 💻 Полноценный RCON

Встроенная консоль позволяет управлять сервером без сторонних программ.

### Возможности

- Вывод команд в реальном времени
- История команд
- Работа сразу с несколькими серверами
- Быстрое выполнение RCON-команд

### Чёрный список команд

Для каждой роли можно отдельно настроить запрещённые команды.

Например:

```
c.*
oxide.*
server.*
global.*
```

По умолчанию критически важные команды доступны только администраторам самого высокого уровня.

---

# 🔒 Защита от злоупотреблений

OverPanel помогает защитить проект даже от недобросовестных сотрудников.

Можно настроить правила, по которым система автоматически:

- снимает права администратора;
- блокирует аккаунт;
- уведомляет владельцев проекта;
- сохраняет событие в журнал аудита.

### Пример

Если администратор выдал слишком большое количество банов, мутов или других наказаний за определённый промежуток времени, система автоматически лишит его прав.

### Массовый откат наказаний

При необходимости можно одной операцией отменить **все наказания**, выданные конкретным администратором за выбранный период времени.

Это особенно полезно при взломе аккаунта администратора или злоупотреблении полномочиями.

- информацию о проектах.

API отлично подходит для интеграции с сайтами, лаунчерами и собственными сервисами.
