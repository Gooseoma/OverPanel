## Что такое OverPanel?

OverPanel — веб-панель для владельцев Rust-серверов и крупных игровых проектов, которая закрывает весь цикл модерации и администрирования: от подключения сервера одной командой до готовой SaaS-инфраструктуры с тарифами.

Интерфейс не перегружен лишними элементами, но при этом даёт всё необходимое для управления серверами, администрацией и игроками — от одного сервера с небольшим сообществом до сети из десятков проектов.

## Ключевые возможности

### 🌍 Мультисервер и мультипроектность

Один аккаунт — несколько проектов, каждый проект — несколько серверов. Подключение нового сервера занимает пару секунд:

```
op.pair <код>
```

После выполнения команды в консоли сервер автоматически появляется в панели. Работает как с **Carbon**, так и с **Oxide/uMod** — менять фреймворк ради панели не нужно.

### 👥 Гибкая система ролей

По умолчанию доступно 10 уровней ролей, но иерархия не жёсткая — можно создавать свои роли с нуля. Каждое право настраивается отдельным тумблером персонально для роли: например, младшему модератору можно точечно выдать доступ, который обычно есть только у топ-админа, не открывая ему всё остальное.

### 🎙 Проверки с голосом

Во время проверки нарушитель видит CUI-оверлей и слышит голосовую озвучку прямо в игре — звук воспроизводится через фейкового игрока, поставленного рядом. Язык (RU/EN) определяется автоматически по клиенту игрока — и для текста, и для аудио.

### 🛡 Свой чит-чекер

Отдельное desktop-приложение сканирует ПК игрока по 30+ категориям: читы, DLL-инжекты, процессы, реестр, автозагрузка Windows, VPN-клиенты, драйверы и другие следы модификаций. Параллельно собирается HWID — это значительно повышает точность определения мультиаккаунтов.

### 🔗 Мультиаккаунты и баны

Панель навсегда запоминает связи между аккаунтами (по IP/HWID). Если на связанном аккаунте есть активная блокировка, срабатывает автобан. Бан-листы можно шарить между разными проектами по подписке — обойти блокировку, просто перейдя на другой сервер сети, не получится.

### 💻 RCON прямо в панели

Полноценная консоль с выводом в реальном времени — без необходимости открывать сторонние RCON-клиенты. Чёрный список команд настраивается по ролям: команды вроде `c.*`, `oxide.*` по умолчанию закрыты для всех, кроме топ-уровня.

### 🚨 Защита от злоупотреблений (антислив)

Правила настраиваются гибко: если админ выдал слишком много банов/мутов за выбранный период, он автоматически теряет права. При необходимости все наказания конкретного администратора за период отменяются массово, одной операцией.

### 📊 Публичный API

API только для чтения — бан-листы, статистика, онлайн-данные можно вывести на внешний сайт, в лаунчер или сторонний сервис, не давая прямого доступа к панели.

## SaaS-тарифы

| Тариф | Для кого |
|---|---|
| **Basic** | Небольшие проекты |
| **Pro** | Активно растущие серверы |
| **Enterprise** | Крупные сети — кастомные лимиты и свой поддомен |

Баланс и автопродление доступны на всех тарифах. Новым пользователям доступен бесплатный пробный период — этого достаточно, чтобы оценить панель на живом проекте.

## Быстрый старт

1. Зарегистрируйте аккаунт и создайте проект в OverPanel.
2. Установите плагин OverPanel на сервер (Carbon или Oxide/uMod).
3. В консоли сервера выполните команду с кодом привязки, полученным в панели:

   ```
   op.pair <код>
   ```

4. Сервер появится в списке проекта — настройте роли, права и модерацию под свою команду.

## Неофициальные сборки сервера

OverPanel рассчитан на работу с официальной версией Rust Dedicated Server. Использование неофициальных, модифицированных или пиратских сборок не запрещено технически, но техническая поддержка в этом случае не предоставляется.
